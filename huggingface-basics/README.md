# Hugging Face Transformers — Basics

## Concept
Hugging Face `transformers` exposes thousands of pre-trained models behind a simple `pipeline` API,
so you can run state-of-the-art NLP (sentiment, classification, generation, QA) in a couple of lines.
It's the foundation of most modern applied NLP.

## Implementation
Hands-on exploration of the `pipeline` API:
- **Sentiment analysis**, **zero-shot classification**, and other tasks on pre-trained models.
- Working with tokenizers and models directly.

A practical introduction to the library that underpins the RAG and chatbot projects elsewhere in this
portfolio.
